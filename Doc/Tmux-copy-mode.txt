
Enter Copy mode:
(ctrl + b) - [


<space>     - start selection
<enter>     - copy selected
<esc>       - clear selection
g           - go to top
G           - go to bottom
h           - move to left
j           - move down
k           - move up
l           - move to right
/           - search
#           - list paste buffer
q           - quit


Past from copy buffer:
(ctrl + b) - ]
